Summary of Residual On-axis Leakage

Scope: manifest indices 14–49, excluding 24–29 (ODC-5241 era; ODC-3611 legacy fields not included).
Each table row is one (ODC weight, calibration variant, reference field) combination, aggregated over all SB_REF observations of that field.
Bandpass calibrated (bpcal) and Bandpass + Leakage on-axis calibrated (lcal) variants are shown in separate tables.
Plot badges on the Reference field column link to per-ODC and all-ODC footprint heatmaps (see legend below). SB_REF media (Stokes/pol.degree MP4, combined-beams PNG, beamwise stats, PAF overlay and movie) are accessible via the drop-down trigger on each SB_REF ID in the Observed SB_REF IDs column.

Column legend

Notation convention: the superscript in parentheses denotes the index (dimension) over which the statistic is computed. For example, \(\mu^{(s)}\) = median over SB_REF observations \(s\); \(\mu^{(b)}\) = median over beams \(b\). Subscripts denote the free indices the result depends on.

All statistics are over percentage linear polarisation leakage, \(dL\) (%). \(dL = 100 \times L/I\), where \(L\) is the linear polarisation intensity and \(I\) is the Stokes I total intensity.

Plot badges on Reference field column:
dL / |dQ|,|dU| (blue) — per-ODC footprint for that row's variant;
dL≈ / |dQ|,|dU|≈ — all-ODC comparison footprint (bpcal + lcal stacked).
dL = residual leakage (dL = √(Q²+U²)/I (%)), faceted by ODC weight and calibration variant.
|dQ|,|dU| = split-circle plots of |Q|/I (top-left) and |U|/I (bottom-right).
